# Cafe Escobar Website

Team project to build e-commerce site for fictional cafe.
Design and implement frontend. 
Built responsive pages with HTML5, CSS3, JavaScript. Created product displays, shopping cart, and checkout forms with client-side validation. Integrated with MongoDB backend via API calls.
Delivered functional storefront with working product browsing and cart management on schedule.

## Features

- Fully responsive design across desktop, tablet, and mobile devices
- Modern UI with clean navigation
- Cross-browser compatibility
- Optimized page load performance
- Menu display with pricing
- Contact information and location details
